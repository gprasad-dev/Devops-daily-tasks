8) pwd : To display present working directory
ex:
gyana@ubuntu:~$ pwd
/home/gyana

9) clear: to clean the screen

10) mkdir : to make directory
syntax:
mkdir foldername(directory name)
ex:
gyana@ubuntu:~$ mkdir testfolder
gyana@ubuntu:~$ ls
testfolder

11) ls : To display the list of the files and directory in pwd
ex:
gyana@ubuntu:~$ ls
testfolder

12) cd : change directory
ex: 
gyana@ubuntu:~$ cd testfolder
gyana@ubuntu:~/testfolder$

13) cd .. : to come back from the directory
ex:
gyana@ubuntu:~$ cd testfolder
gyana@ubuntu:~/testfolder$ cd ..
gyana@ubuntu:~$

14) rmdir : To remove empty directory
syntax: 
rmdir directory_name
ex:
gyana@ubuntu:~$ rmdir testfolder
gyana@ubuntu:~$ ls
gyana@ubuntu:~$	---> This is showing empty because the one directory(testfolder) we had created has been removed.

15) rm -r dir : delete the directory if it is not empty
ex:
gyana@ubuntu:~$ mkdir gyana_files
gyana@ubuntu:~$ ls
gyana_files
gyana@ubuntu:~$ rm -r gyana_files
gyana@ubuntu:~$ ls
gyana@ubuntu:~$ ---> The directory has been deleted

16) mv : to rename the file and directory
syntax:
"mv file's_old_name new_name"
ex:
gyana@ubuntu:~$ mv testfolder user_folder
gyana@ubuntu:~$ ls
gyana_files  user_folder

17)touch : to create empty files
syntax:
"touch file_name"
ex:
gyana@ubuntu:~$ touch text
gyana@ubuntu:~$ ls
gyana_files  text  user_folder

18)rm : to remove the empty file
	It can't remove the folder only can remove the file
syntax:
"rm file_name"
ex: 
gyana@ubuntu:~$ touch text.txt
gyana@ubuntu:~$ ls
gyana_files  text.txt  user_folder
gyana@ubuntu:~$ rm text.txt
gyana@ubuntu:~$ ls
gyana_files  user_folder

19) cat : case-1 : It will  create a file with data
syntax:
cat > file_name
Enter the data use then press ctrl+d to save the data inside file. If we press "enter" the it will go to next line/row.
ex:
gyana@ubuntu:~$ touch text.txt
gyana@ubuntu:~$ ls
gyana_files  text.txt  user_folder
gyana@ubuntu:~$ cat > text.txt
Used 'cat' command to enter data in the text.txt file.
gyana@ubuntu:~$ cat text.txt
Used 'cat' command to enter data in the text.txt file.
gyana@ubuntu:~$
gyana@ubuntu:~$ cat > text.txt
Testing the cat command
Hit enter button to shift to next line.
gyana@ubuntu:~$ ls
gyana_files  text.txt  user_folder
gyana@ubuntu:~$ cat text.txt			(It override the file content)
Testing the cat command
Hit enter button to shift to next line.   
gyana@ubuntu:~$
